# Rooftop Segmentation - Training Notebook

This branch contains the Jupyter notebook used to train the U-Net rooftop
segmentation model that powers Module 02 of the Solarys platform.

## Files
- rooftop_segmentation_baseline.ipynb - full pipeline (dataset loading,
  data augmentation, U-Net architecture, training with Dice + IoU,
  export to .keras + meta.json).

## Dataset
Kaggle: slyveinweeb/rooftop-images-for-semantic-segmentation
1671 image/label pairs, binary roof segmentation.
